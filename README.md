# FinSight (CashMap)

가계부 AI 분석 및 소비 패턴 관리 서비스입니다.

🚀 **실제 배포 링크 (Vercel)**: [https://finsightdongjun.vercel.app/](https://finsightdongjun.vercel.app/)

## 🛠️ 기술 스택
- **Frontend**: Next.js (App Router), TailwindCSS, TanStack React Query, Recharts
- **Database**: Supabase PostgreSQL, Prisma ORM
- **Deployment**: Vercel

## 📁 프로젝트 구조
- `/web`: Next.js 기반 웹 애플리케이션
- `/backend`: 백엔드 관련 리소스 (배포 빌드 타겟 등)

## 🚀 시작하기 (Web Client)

```bash
cd web
npm install
npm run dev
```
